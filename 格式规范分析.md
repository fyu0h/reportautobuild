### 5️⃣ 一级标题（章节标题）

|------|-----|

| 西文字体 | **黑体** |

| 首行缩进 | **2字符**（1.13cm） |

---

### 6️⃣ 正文段落（新闻条目）

正文段落包含**两部分不同格式**：

#### 导语（首句）— 加粗

| 属性 | 值 |
|------|-----|
| 示例 | `1.港将公布《香港铁路标准》推湾区应用。` |
| 中文字体 | **仿宋_GB2312** |
| 西文字体 | Times New Roman |
| 字号 | **16磅（三号）** |
| 加粗 | ✅ **是** |

#### 正文内容 — 不加粗

| 属性 | 值 |
|------|-----|
| 示例 | `据...X月X日报道，.......。（皇岗边检站）` |
| 中文字体 | **仿宋_GB2312** |
| 西文字体 | Times New Roman |
| 字号 | **16磅（三号）** |
| 加粗 | ❌ 否 |

> [!TIP]
> 每条新闻的**标题部分（至第一个句号）加粗**，后续正文内容不加粗。末尾用括号标注信息来源，如 `（皇岗边检站）`。

## 三、整体格式规范总结

| 规范项 | 统一设置 |
|--------|----------|
| 全文行距 | **固定值 28磅** |
| 正文字号 | **三号（16磅）** |
| 正文字体 | **仿宋_GB2312** |
| 西文字体 | **Times New Roman** |
| 首行缩进 | **2字符**（除标题外） |
| 对齐方式 | 标题居中，正文两端对齐 |
| 大标题字体 | **方正小标宋简体，二号（22磅）** |
| 副标题字体 | **楷体_GB2312，三号（16磅）** |
| 一级标题字体 | **黑体，三号（16磅）** |
| 新闻首句 | **加粗** |
